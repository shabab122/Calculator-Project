# Modern Gradient Calculator

A stylish and responsive calculator built with HTML, CSS, and JavaScript. This project combines basic calculator functionality with a modern glassmorphism-inspired design, colorful gradient buttons, and smooth user interactions to create a visually appealing user experience.

## Features

* Perform basic arithmetic operations (+, -, ×, ÷)
* Percentage (%) calculation support
* Clear (C) and Delete (DEL) functions
* Decimal number calculations
* Modern gradient-based UI design
* Glassmorphism calculator container
* Responsive layout for different screen sizes
* Smooth hover and click animations
* Clean and beginner-friendly code structure

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How It Works

The calculator captures user input through interactive buttons and displays the expression on the screen. JavaScript processes the entered expression and instantly calculates the result when the equals (=) button is pressed. Additional functions allow users to clear the display or remove the last entered character.

## Why This Project?

Most beginner calculator projects focus only on functionality. This project goes a step further by combining functionality with a modern and attractive user interface. It demonstrates how simple web technologies can be used to create an application that is both useful and visually engaging.
